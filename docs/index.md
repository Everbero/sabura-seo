## Diretrizes importantes para o layout (SEO)

Os conteúdos deste projeto não são apenas textos - eles já foram estruturados estrategicamente para SEO.  
Por isso, o layout deve respeitar essa estrutura para não comprometer o desempenho orgânico.

### Hierarquia de títulos

- O **H1 é único por página**
- Os **H2 organizam os blocos principais**
- Os **H3 detalham subseções quando necessário**
- A ordem dos títulos **não deve ser alterada**

Essa hierarquia é essencial para indexação e compreensão do conteúdo pelo Google.

---

### Listas (extremamente importantes)

- Listas (`ul/li`) **devem permanecer como listas**
- **Não transformar listas em parágrafos**
- **Não adicionar ponto final em itens de lista**

Motivo: listas são frequentemente usadas pelo Google para gerar **featured snippets** e melhorar a leitura.

---

### Estrutura de conteúdo

- Evitar juntar parágrafos
- Evitar transformar blocos em textos longos
- Manter separação entre seções
- Respeitar a ordem original do conteúdo

A organização atual foi pensada para facilitar leitura e escaneabilidade.

---

### Adaptação visual

O layout pode (e deve) trabalhar:

- Destaque visual de seções
- Uso de espaçamento e respiro
- Componentes visuais (cards, blocos, etc.)
- Tipografia e hierarquia visual

Mas **sem alterar a estrutura semântica do HTML** (títulos, listas e ordem do conteúdo).

## Objetivo

Garantir que o site una:

- Clareza de comunicação  
- Qualidade visual  
- Performance em SEO  

Qualquer alteração estrutural deve ser avaliada considerando impacto em ranqueamento.
